# ML
Demo Machine Learning

## Web UI (Streamlit)

Untuk kemudahan penggunaan, proyek ini dilengkapi dengan antarmuka web:

```bash
streamlit run app.py
```

Kemudian buka browser di `http://localhost:8501` 

Fitur:
- Upload CSV untuk training
- Konfigurasi model via UI
- Visualisasi hasil training
- Upload CSV untuk prediksi
- Download hasil prediksi
